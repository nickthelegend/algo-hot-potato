# 🥔 AlgoPotato — The Hot Potato Game on Algorand

**AlgoPotato** is a decentralized game of **Hot Potato** , where two players stake an equal amount of Algo or an Algorand Standard Asset (ASA), and the winner takes it all—thanks to a provably fair randomness mechanism using **Verifiable Random Functions (VRF)** .

This game runs entirely on Algorand smart contracts and is accessible through a website interface where players can share links to challenge others and join games.

## 🎮 Game Overview

### 💰 Game Flow

1. **Game Creation**
   * A player deposits Algo or ASA into a new game box.
   * Game details (player address, asset type, amount) are saved on-chain.
   * A **VRF Round** is selected — 9 rounds into the future.
2. **Game Join**
   * A second player joins by depositing the **same asset and amount** .
   * The game is now ready to be triggered using the VRF randomness.
3. **VRF Determination**
   * When the selected VRF round is reached, a random number is generated.
   * This number determines how many "passes" occur between the players.
   * The last player holding the "hot potato" **loses** , and the winner takes all.
4. **Reward Distribution**
   * The contract pays out the full amount to the winner.
   * The loser receives nothing.

### 🚨 Special Conditions

* If **no second player joins** , the first player can **cancel** and get their deposit back.
* If the selected VRF round is **too old** (older than 1,512 rounds), it resets to 9 rounds in the future.
* If either player **opts out of the ASA** , the game will:
  * Reward the remaining opted-in player.
  * Stay live if both are opted out, waiting for at least one to opt back in.

---

## 🔒 Fairness via VRF

* **Randomness Beacon** provides VRF output from the Algorand network.
* Output is converted to a `uint256` and then cast to a `uint64` using modulo 240:
  * `passes = uint256(vrf_output) % 240`
* **Why 240?**
  * Algorand supports a max of 256 inner transactions.
  * 14 are used for opup budget.
  * 1 is the VRF call.
  * 1 for final reward payment.
